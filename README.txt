--INSTALACION--

1- BASE DE DATOS
PARA EL PROYECTO USAMOS LA BASE DE DATOS MYSQL DESDE XAMPP. 
SE DEBE CREAR UNA BASE DE DATOS CON EL NOMBRE hoteles. DENTRO DE LA CARPETA LLAMADA bdd
SE ENCUENTRA UN ARCHIVO LLAMADO habitaciones.sql IMORTAR DICHO ARCHIVO EN LA BASE DE DATOS
MENCIONADA ANTERIORMENTE

DENTRO DEL ARCHIVO LLAMADO config.py SE ENCUENTRAN LOS DATOS DE LA CONEXION, SE DEBE CAMBIAR
EL NOMBRE DEL HOST, USER O PASSWORD EN CASO DE SER NECESARIO, EN NUESTRO CASO TENEMOS TODO CONFIGURADO POR DEFECTO

2-ABRIR EL PROYECTO EN UN EDITOR DE TEXTO
3-CREAR UN ENTORNO VIRTUAL CON EL COMANDO "virtualenv -p python3 env"
4-ACTIVAR EL ENTORNO VIRTUAL CON EL COMANDO ".\env\Scripts\activate"
5-INSTALAR TODAS LAS DEPENDENCIAS MENCIONADAS EN EL ARCHIVO LLAMADO requerimientos.txt 

--DESPLIEGE Y FORMA DE USO DE LA APLICACION--
AL EJECUTAR EL ARCHIVO PRICIPAL app.py LA APLICACION DE FLASK CORRERA EN EL SERVIDOR WAITRESS QUE CORRE EN EL
PUERTO localhost:8080
LA VISTA Y LA RUTA PRINCIPAL ES /. A PARTIR DE ESTO SE VERIFICARÁ SI EL USUARIO ESTA LOGEADO.
EN LA RUTA RAIZ SE DECIDIRA SI SE QUIERE INGRESAR CON LAS CREDENCIALES CARGADAS PREVIAMENTE EN EN LA BDD hoteles EN LA
TABLA login, O BIEN SI SE QUIRE REGISTRAR UN NUEVO USAURIO ELIGIENDO LOS PERMISOS QUE TENDRÁ EL MISMO
SE DEBE INGRESAR A CADA RUTA CON EL NOMBRE INDICADO PARA REALIZAR LOS REQUEST, ADEMAS DEPENDIENDO DE LA ACCION QUE
SE DESEA REALIZAR, SE DEBEN AGREGAR LOS PARAMETROS DETALLADOS EN CADA METODO (SE ENCUENTRA COMENTADO EN CADA SECCION
DEL CODIGO).
